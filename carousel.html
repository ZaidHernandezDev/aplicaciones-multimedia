Lenguajes de Programación

Se destacan lenguajes como Processing, OpenGL y SVG, que permiten desarrollar aplicaciones con elementos visuales y gráficos interactivos. Web: Se mencionan HTML, CSS, JavaScript, PHP y Python, que son esenciales para crear y gestionar sitios web dinámicos y aplicaciones web interactivas.

Entornos de desarrollo integrados(IDE)

Se describen herramientas que ofrecen todas las herramientas necesarias para el desarrollo en un solo programa, como editores de código y depuradores.

Frameworks de desarrollo web

Se incluyen React, Django y Angular, que simplifican el desarrollo de aplicaciones web proporcionando una estructura estándar.

Ambientes de desarrollo para dispositivos móviles

Se describen herramientas y entornos específicos para optimizar aplicaciones para smartphones y tablets, tanto en desarrollo nativo como multiplataforma.

Arquitectura de las aplicaciones multimedia

Se explican diferentes modelos arquitectónicos, como la arquitectura cliente-servidor, en la nube, de microservicios, basada en eventos e híbrida.
